    Models Reference¶

    swb.models ¶

    Data models for SWB API.

    Classes¶

    DatabaseInfo dataclass ¶

    Information about the SRU database.

    Attributes:

    Name Type Description
    title str

    Database title
    description str | None

    Database description
    contact str | None

    Contact information

    ExplainResponse dataclass ¶

    Represents the response from an SRU explain operation.

    The explain operation provides information about the server's capabilities, including available search indices, supported record formats, and database information.

    Attributes:

    Name Type Description
    server_info ServerInfo

    Information about the server
    database_info DatabaseInfo

    Information about the database
    indices list[IndexInfo]

    List of available search indices
    schemas list[SchemaInfo]

    List of supported record schemas

    IndexInfo dataclass ¶

    Information about a searchable index.

    Attributes:

    Name Type Description
    title str

    Index title/name
    name str

    Index identifier (CQL name)
    description str | None

    Optional description of the index

    LibraryHolding dataclass ¶

    Represents library holding information for a record.

    Attributes:

    Name Type Description
    library_code str

    Library identifier code (e.g., DE-21, DE-15)
    library_name str | None

    Human-readable library name
    access_url str | None

    URL for accessing the resource at this library
    access_note str | None

    Note about access restrictions or requirements
    collection str | None

    Collection or database name

    RecordFormat ¶

    Bases: str, Enum

    Supported record formats for SRU queries.

    RecordType ¶

    Bases: str, Enum

    Record types for filtering searches.

    RelationType ¶

    Bases: str, Enum

    Record relationship types for band/linking search.

    Used to find related publications in multi-volume works.

    ScanResponse dataclass ¶

    Represents the response from an SRU scan operation.

    Attributes:

    Name Type Description
    terms list[ScanTerm]

    List of scan terms
    scan_clause str

    Original scan clause
    response_position int

    Position in the term list

    ScanTerm dataclass ¶

    Represents a single term from a scan operation.

    Attributes:

    Name Type Description
    value str

    The term value
    number_of_records int

    Number of records containing this term
    display_term str | None

    Human-readable display form of the term
    extra_data str | None

    Additional term metadata

    SchemaInfo dataclass ¶

    Information about a record schema format.

    Attributes:

    Name Type Description
    identifier str

    Schema identifier (e.g., "marcxml", "mods")
    name str

    Human-readable schema name
    title str | None

    Optional schema title

    SearchIndex ¶

    Bases: str, Enum

    Available search indices for CQL queries.

    SearchResponse dataclass ¶

    Represents the complete response from an SRU search query.

    Attributes:

    Name Type Description
    total_results int

    Total number of results found
    results list[SearchResult]

    List of individual search results
    next_record int | None

    Position of the next record for pagination
    query str

    Original CQL query string
    format RecordFormat

    Record format used
    Attributes¶
    has_more property ¶

    Check if there are more results available.

    SearchResult dataclass ¶

    Represents a single search result from the SWB API.

    Attributes:

    Name Type Description
    record_id str | None

    Unique identifier for the record
    title str | None

    Title of the resource
    author str | None

    Author(s) of the resource
    year str | None

    Publication year
    publisher str | None

    Publisher information
    isbn str | None

    ISBN number(s)
    raw_data str | None

    Raw XML data from the API
    format RecordFormat

    Format of the record data
    holdings list[LibraryHolding] | None

    List of library holdings for this record
